# 📰 Fake News Detection using Machine Learning

An end-to-end Fake News Detection web application built using Natural Language Processing (NLP), Machine Learning, and Streamlit. The application predicts whether a news article is **Real** or **Fake** based on its textual content.

---

## 🚀 Features

- Predicts whether news is **Real** or **Fake**
- NLP preprocessing pipeline
- Text cleaning and stemming
- TF-IDF Vectorization
- Interactive Streamlit web interface
- Confidence score visualization
- Fast real-time predictions

---

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/Fake-News-Detection.git
```

Move into the project folder

```bash
cd Fake-News-Detection
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
